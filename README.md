# AgenticAuthor

AI-powered iterative book generation with natural language feedback and git-backed version control.

## Quick Start

```bash
# Install
pip install -e .

# Set API key
export OPENROUTER_API_KEY="sk-or-your-key-here"

# Run
agentic
```
