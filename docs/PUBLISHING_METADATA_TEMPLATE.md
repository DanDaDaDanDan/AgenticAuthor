## Book Description

Your book description is your sales pitch on Amazon. This appears on your book's product page.

**Requirements**:
- Length: 100-150 words (~500-750 characters)
- Format: HTML allowed (`<b>`, `<i>`, `<br>`)
- No spoilers, reviews, or contact info

**Description**:

```
[Generated description will appear here]

Structure:
- Hook (1-2 bold sentences to grab attention)
- Main conflict and protagonist
- Stakes (what's at risk?)
- Call to action

Use HTML tags:
<b>Bold for emphasis</b>
<i>Italic for titles</i>
<br> for line breaks
```
