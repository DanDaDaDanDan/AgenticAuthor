## 2. Book Description (Sales Copy)

**Character Limit**: 4,000 characters maximum
**Optimal Length**: 100-150 words (~500-750 characters)
**Current Length**: 0 characters

### Writing Guide
- Start with attention-grabbing hook
- Present main conflict/problem
- Include emotional stakes
- Integrate keywords naturally
- Use HTML formatting (bold, italics, line breaks)
- End with call to action

### Description Text

```
[Write your 4,000-character book description here]

Structure for Fiction:
- Hook (1-2 sentences)
- Introduce protagonist and conflict (2 paragraphs)
- Raise stakes
- Call to action

Structure for Nonfiction:
- Problem statement (reader's pain point)
- Your solution (what book offers)
- Your credentials (why you're qualified)
- Benefits (what reader will gain)
- Call to action

Use HTML for formatting:
<b>Bold for emphasis</b>
<i>Italics for titles or emphasis</i>
<br> for line breaks

Avoid:
- Reviews/testimonials
- Contact information
- Spoilers
- Time-sensitive info
```

- [ ] Finalize description and keywords
- [ ] Research categories and competition
- [ ] Create author bio and update website
- [ ] Design cover (professionally done)
- [ ] Get beta readers
- [ ] Build ARC (Advanced Review Copy) team

**Launch Week**:
- [ ] Set up Amazon Author Central page
- [ ] Claim Goodreads author profile
- [ ] Announce on social media
- [ ] Email mailing list
- [ ] Reach out to book bloggers/reviewers
- [ ] Consider BookBub Featured Deal submission
- [ ] Monitor sales rank and reviews

**Post-Launch (ongoing)**:
- [ ] Respond to reviews (appropriately)
- [ ] Update keywords based on performance
- [ ] Adjust price if needed
